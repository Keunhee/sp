OctaFlip 게임 규칙 분석:

기존 hw2_202211195.c 코드를 분석한 결과, OctaFlip의 게임 규칙은 다음과 같습니다:

1. 게임 보드:
   - 8x8 크기의 보드
   - 초기 상태: 왼쪽 상단(0,0)과 오른쪽 하단(7,7)에 'R' 말, 오른쪽 상단(0,7)과 왼쪽 하단(7,0)에 'B' 말 배치

2. 이동 규칙:
   - 플레이어는 자신의 말을 상하좌우 또는 대각선 방향으로 1칸 또는 2칸 이동 가능
   - 2칸 이동 시, 중간 칸은 비어있어야 함 (R이나 B가 있으면 안 됨)
   - 이동 목적지는 빈칸('.')이어야 함
   - 2칸 이동 시 원래 위치의 말은 제거됨(board[r1][c1]='.')

3. 말 뒤집기 규칙:
   - 이동 후 인접한(상하좌우 및 대각선) 상대편 말은 모두 자신의 말로 변경됨

4. 게임 종료 조건:
   - 한 플레이어의 말이 모두 사라진 경우
   - 보드가 가득 찬 경우 (빈칸이 없는 경우)
   - 연속으로 두 번 패스한 경우 (양쪽 모두 유효한 이동이 없는 경우)
   - 지정된 이동 횟수(moveCount)에 도달한 경우

5. 승패 결정:
   - 게임 종료 시 Red(R) 말이 더 많으면 Red 승리
   - Blue(B) 말이 더 많으면 Blue 승리
   - 동일한 경우 무승부(Draw)

6. 패스 규칙:
   - 유효한 이동이 없는 경우 패스 가능 (0 0 0 0 입력)
   - 유효한 이동이 있는데 패스하면 무효한 이동으로 간주
